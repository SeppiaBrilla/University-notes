N-gram language models

The N-gram model is a probabilistic model. Ideally, we would base the probability of the next word on all the previous ones, but the probability table would be too big to be store, therefore, we need to limit the model to a set of words. If the model is based on none previous words, is called unigram while the one based on one previous word is the bigram. All the N-gram models are based on the Naive Bayes classifier .
