LDA

LDA is a supervised ML algorithm that aim to reduce dimensions (just like PCA ) but also wants to keep the maximum difference between cluster of data so that is easier to visualize the clusters. The main difference is on how the data is centered: not by the center of all data but only by the center of each data cluster so that each elements spans only around the his cluster. Implementation:
