cholesky factorization for linear least-sqare problem

The Cholesky factorization for the lls problem is a direct method to find the solution.

  1. we can rewrite the equation as: ATA=ATbA^TA = A^TbATA=ATb
  2. we can factorize ATAA^TAATA with the Cholesky factorization in ATA=LLTA^TA = LL^TATA=LLT where LLL is a lower triangular matrix. No pivoting is needed and this factorization always exist for ATAA^TAATA symmetric positive definite
  3. LLT=ATb⇒Ly=ATbLL^T = A^Tb \Rightarrow Ly = A^TbLLT=ATb⇒Ly=ATb, then LTx=yL^Tx = yLTx=y

The computational costs is: O(n36)O(\frac{n^3}{6})O(6n3​)
