Turing machine

Turing Machine (TM for short) working on k tapes is described as a triple (Γ,Q,δ)(\Gamma, Q ,\delta)(Γ,Q,δ) containing:

  • Γ\GammaΓ: a finite alphabet, always including two character ⊳,□\rhd, \Box⊳,□ representing respectively initial and blank characters and 0,1
  • QQQ: a fine set of possible states including two special states which indicates the initial and final states.
  • δ:Q×Γk→Q×Γk×{L,S,R}k\delta: Q \times \Gamma^k \rightarrow Q \times \Gamma^k \times \{L,S,R\}^kδ:Q×Γk→Q×Γk×{L,S,R}k: a transition function

A configuration, C, contains: the current state q, the content of the k tapes and the position of the k tape heads.
