Ant colony optimization

Ant colony optimization is a Swarm Intelligence algorithm. Any individual leave behind him some sort of "pheromone" that the other individuals will follow. Uses a probabilistic parameterized model (here some notes on probabilistic models ) to model pheromone trails and the solution is build in an incremental way with stochastic steps in a fully connected graph called construction graph G = (C,L) with C being vertexes / solution components, L arcs / connections and States being path on G.

Pseudocode:

def ACO():
    initializePheromonesValues()
    while termination_conditions:
        for ant in ants:
            solution_a = buildSolution(pheromones, heuristic)
        applyOnlineDelayedPheromoneUpdate()
