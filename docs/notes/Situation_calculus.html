Situation calculus

It is a Deductive planning technique that aims at resolving a Planning problem as a First order logic deduction. It maps the world in states and the first one is called situation. In order to resolve the Frame problem, we add to each state an extra axiom: Ft+1⇔Actiont∧(Ft∧¬actionthatcausesnotF)F^{t+1} \Leftrightarrow Action^t \wedge (F^t \wedge \neg \text{actionthatcausesnotF})Ft+1⇔Actiont∧(Ft∧¬actionthatcausesnotF)
