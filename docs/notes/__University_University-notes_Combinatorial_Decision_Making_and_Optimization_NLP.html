NLP

It stands for Non-Linear Programming, which, is like Linear programming (thus, also an OR ) but with non-linear functions like y=x2+1zy = x^2 + \frac{1}{z}y=x2+z1​. In order to solve this kind of problems, we can operate in two different ways:

  • Encode the problem in an equally satisfiable linear problem (From NLP to LP )
  • Use a specific NLP approach
