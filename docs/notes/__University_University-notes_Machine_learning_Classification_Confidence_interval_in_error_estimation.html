Confidence interval in error estimation

We can forecast each element of the test set like in a Bernulli experiment process:

  • Good prediction -> success
  • Bad prediction -> failure The empirical frequency of an error is SN\frac{S}{N}NS​ where N is the number of independent experiment made (number of elements of the test set). The deviation of the empirical frequency from the true frequency is due to noise. Noise is assumed to have a normal distribution around true probability
