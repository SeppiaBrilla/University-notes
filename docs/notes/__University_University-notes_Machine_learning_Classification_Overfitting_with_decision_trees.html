Overfitting with decision trees

A Decision tree can be defined as a set of hypothesis (h) of the relationship between the predictor attributes and the class. A decision tree overfits with a set of hypotesis when we can find a new set of hypotesis (h') such as: errortrain(h)<errortrain(h′)errorX(h)>errorX(h′) \begin{align} \text{error}_{\text{train}}(h) &< \text{error}_{\text{train}}(h')\\ \text{error}_{\mathcal{X}}(h) &> \text{error}_{\mathcal{X}}(h')\\ \end{align} errortrain​(h)errorX​(h)​<errortrain​(h′)>errorX​(h′)​​
