Sensitivity analysis

Sensitivity analysis refers to how the optimal solution of a problem (Linear programming or Operations research ) is affected by changes in the input parameters. If x* is the optimal solution for standard LP problem P, will x* be still feasible and/or optimal for a perturbed problem?

Changing value

Cost coefficients: Changing c→c=c+Δcc \rightarrow c = c + \Delta cc→c=c+Δc can’t affect feasibility, but may involve:

  • Loss of optimality
  • Different objective value for current solution (still optimal)
