Define a model for learning process in AI

A model in artificial intelligence is a function that given some data predicts an outcome. A model could be:

  • A deterministic function
  • A probabilistic model

The creation of a model is divided in two steps:

  • Training step: where we train our model on the data we have and tune it to predict on that data
  • Testing step: where we use some other data to test how good is our predicting model on unseen data Generally, we split our dataset in two dataset: Train and Test each used in the respective step
