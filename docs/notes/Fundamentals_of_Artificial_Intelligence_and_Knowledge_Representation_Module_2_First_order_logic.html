First order logic

It is a kind of Fundamentals of Artificial Intelligence and Knowledge Representation/Module 2/Logic built upon Proposition logic . Given this, we can define FOL as:

  • The set of constants (always true or always false)
  • The set of function symbols (e.g. appear_on(film) ≡\equiv≡ TV)
  • The set of variables
  • The set of predicate symbols
  • The set of logical connectives (as in proposition logic but with ∃  and  ∀\exists \; \text{and} \; \forall∃and∀ included)

Way of reasoning

We now need a way to decide if a formula is true or not. In order to do so, we can introduce the concept of engine.

Engine

an engine is a method of reasoning on a logic formula in order to assign to that formula a value (T o F). We write F1∧⋯∧Fn⊣EGF_1 \wedge \dots \wedge F_n \dashv^E GF1​∧⋯∧Fn​⊣EG to denote that the formula G is deducible from F1∧⋯∧FnF_1 \wedge \dots \wedge F_nF1​∧⋯∧Fn​ with the engine E.

Sound

A method E is said to be sound if holds: Γ⊣Eφ⇒Γ⊨φ\Gamma \dashv^E \varphi \Rightarrow \Gamma \models \varphiΓ⊣Eφ⇒Γ⊨φ.

Complete

A method E is said to be sound if holds: Γ⊨φ⇒Γ⊣Eφ\Gamma \models \varphi \Rightarrow \Gamma \dashv^E \varphiΓ⊨φ⇒Γ⊣Eφ.

Reasoning by deduction

Theorem: Given a set of formulas {F1,…,Fn}\{F_1, \dots, F_n\}{F1​,…,Fn​} and a formula G, F1∧⋯∧Fn⊨GF_1 \wedge \dots \wedge F_n \models GF1​∧⋯∧Fn​⊨G if and only if ⊨(F1∧⋯∧Fn)→G\models (F_1 \wedge \dots \wedge F_n) \rightarrow G⊨(F1​∧⋯∧Fn​)→G. The proof proceed by absurd, proving that F1∧⋯∧Fn∧¬GF_1 \wedge \dots \wedge F_n \wedge \neg GF1​∧⋯∧Fn​∧¬G is inconsistent.
